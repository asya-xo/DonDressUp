<br>**name:** DON dressup 🩷 
<br>**what is it:** a dress up webbrowser game 
<br>**made with:** javacript (code), phaser (game engine), canva (buttons, signs), krita (clothing assets + model) 
<br>**lore:** made for our little cousin who loves fashion very much
<br>**more lore:** our cousin drew the clothing assets on paper, and we drew them digitally (or tried to)

<br>**try it out!**
<br>**prerequisites:**
- just a modern web browser
- a local dev server (vs code live server perhaps? or npx serve, up to you)

<br>**installation**
<br>clone the repo:
<br>  git clone https://github.com/your-username/don-dress-up.git
<br>   cd don-dress-up

<br>start a local server: 
<br>  npx serve .

<br> open http://localhost:3000 on ur browser
